# Student Result Analyzer

## Overview
Student Result Analyzer is a Python project that reads student data from a CSV file, calculates total marks, average marks, assigns grades, identifies the top performer, and visualizes the results using charts.

## Features
- Read student data from a CSV file
- Calculate Total Marks
- Calculate Average Marks
- Assign Grades
- Find Top Performer
- Display Student Result
- Display Bar Chart
- Display Pie Chart

## Technologies Used
- Python
- Pandas
- Matplotlib
